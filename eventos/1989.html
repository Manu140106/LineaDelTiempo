1989 - World Wide Web (WWW)

La Invención que Cambió el Mundo


Tim Berners-Lee - El padre de la World Wide Web

¿Qué es la World Wide Web?

La World Wide Web (WWW o simplemente "la Web") es un sistema de información que permite acceder a documentos y recursos a través de Internet mediante hipervínculos. No es lo mismo que Internet (que es la infraestructura de redes), sino una aplicación que funciona sobre Internet.

Internet ≠ World Wide Web

Internet: La red de redes (infraestructura)

WWW: Sistema de páginas web interconectadas

El Creador: Tim Berners-Lee

Sir Tim Berners-Lee, un científico británico, inventó la World Wide Web en 1989 mientras trabajaba en el CERN (Organización Europea para la Investigación Nuclear) en Suiza. Su objetivo era crear un sistema para que los científicos pudieran compartir información fácilmente.

Datos sobre Tim Berners-Lee:

  • Nació el 8 de junio de 1955 en Londres, Inglaterra
  • Trabajaba como ingeniero de software en el CERN
  • Graduado en Física de la Universidad de Oxford
  • Recibió el título de caballero (Sir) por la Reina Isabel II en 2004
  • Fundó la World Wide Web Foundation

La Propuesta Original

En marzo de 1989, Tim Berners-Lee presentó un documento titulado "Information Management: A Proposal" a su jefe en el CERN. Inicialmente, su jefe escribió en el documento: "Vago, pero interesante". Sin embargo, le dio permiso para continuar trabajando en la idea.

Dato curioso: Tim Berners-Lee originalmente quería llamar a su invención "Mesh" (Malla) o "Mine of Information" (Mina de Información), pero finalmente eligió "World Wide Web".


Las Tres Tecnologías Fundamentales

Tim Berners-Lee inventó tres tecnologías que siguen siendo la base de la Web moderna:

Tecnología Nombre Completo Función
HTML HyperText Markup Language Lenguaje para crear páginas web
HTTP HyperText Transfer Protocol Protocolo para transmitir páginas web
URL Uniform Resource Locator Direcciones web (ej: http://ejemplo.com)

HTML - El Lenguaje de las Páginas Web

HTML es el lenguaje que se usa para crear páginas web. Utiliza etiquetas para dar estructura y formato al contenido.

Ejemplo Simple de HTML:

<html>
  <head>
    <title>Mi Primera Página</title>
  </head>
  <body>
    <h1>¡Hola Mundo!</h1>
    <p>Esta es mi primera página web.</p>
  </body>
</html>

Los Hipervínculos: La Magia de la Web

La característica más revolucionaria de la WWW fueron los hipervínculos (o hiperenlaces). Permiten saltar de una página a otra con un simple clic, creando una "telaraña" de información interconectada.

¿Cómo Funcionan los Hipervínculos?

  1. Una palabra o imagen en una página está "enlazada" a otra página
  2. Cuando haces clic en el enlace, el navegador solicita la nueva página
  3. El servidor envía la página solicitada
  4. Tu navegador muestra la nueva página

Ejemplo: <a href="https://ejemplo.com">Haz clic aquí</a>


El Primer Navegador Web

Tim Berners-Lee también creó el primer navegador web en 1990, llamado simplemente "WorldWideWeb" (más tarde renombrado a "Nexus" para evitar confusión). Este navegador también funcionaba como editor, permitiendo crear y modificar páginas web.

Característica Detalle
Año de la Propuesta Marzo de 1989
Creador Tim Berners-Lee
Lugar CERN, Ginebra, Suiza
Primer Navegador WorldWideWeb (1990)
Primera Página Web 6 de agosto de 1991
Objetivo Original Compartir información científica

Una Decisión Revolucionaria

En 1993, el CERN tomó una decisión histórica: hizo que la tecnología WWW fuera de dominio público, es decir, libre y gratuita para todos. Tim Berners-Lee insistió en que no debía haber patentes ni regalías. Esta decisión permitió que la Web se expandiera por todo el mundo.

💡 Frase de Tim Berners-Lee:
"Esta es para todos"
(Mensaje que escribió durante la apertura de los Juegos Olímpicos de Londres 2012)


Impacto y Crecimiento

La World Wide Web transformó Internet de una herramienta para expertos en computación a algo accesible para todos. Facilitó la creación de contenido, el comercio electrónico, las redes sociales y prácticamente toda la actividad en línea que conocemos hoy.

Crecimiento de la Web:

Año Número de Sitios Web
1991 1 sitio web
1994 3,000 sitios web
2000 17 millones
2010 200 millones
2024 Más de 1,800 millones

El Legado de Tim Berners-Lee

Tim Berners-Lee continúa trabajando para mejorar la Web. Fundó el W3C (World Wide Web Consortium) en 1994 para desarrollar estándares web. Hoy en día, aboga por una web abierta, neutral y descentralizada.

Reconocimientos:

  • Caballero del Imperio Británico (2004)
  • Premio Turing (2016) - el "Nobel de la Computación"
  • Miembro de la Orden del Mérito (2007)
  • Medalla de Oro del Instituto de Ingenieros Eléctricos y Electrónicos
