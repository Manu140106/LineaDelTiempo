Eventos Históricos del Internet

1969

ARPANET - El Nacimiento de Internet

Primera red de computadoras que dio origen a Internet. Conectó 4 universidades estadounidenses.

1971

Primer Correo Electrónico

Ray Tomlinson envía el primer email e introduce el símbolo @.

1983

Protocolo TCP/IP

ARPANET adopta TCP/IP. Nacimiento oficial de Internet moderno.

1984

Sistema DNS

Creación de dominios .com, .org y .edu.

1989

World Wide Web

Tim Berners-Lee propone la WWW.

1991

Primera Página Web

Se publica la primera página web en el CERN.

1993

Navegador Mosaic

Primer navegador gráfico que popularizó Internet.

1995

Internet Comercial

Nacen Amazon y eBay, inicia el e-commerce.

2000

Redes Sociales

Surgen plataformas sociales modernas.

2004

Web 2.0

Internet interactivo, usuarios crean contenido.

2010

Internet Móvil

Los smartphones impulsan el acceso móvil.

2020

Educación Virtual

Internet esencial para trabajo remoto y educación.
